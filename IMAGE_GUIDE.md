## Image Optimization Checklist

Before adding images to `/public/images/`:
- [ ] Resize to recommended dimensions
- [ ] Compress using TinyPNG or similar (aim for <300KB per image)
- [ ] Convert to WebP format for better performance
- [ ] Use descriptive filenames (lowercase, dashes)
- [ ] Ensure images are sharp and well-lit
- [ ] Verify commercial use rights

## Tools for Image Optimization

1. **TinyPNG** - https://tinypng.com
   - Compress JPG/PNG files
   - Maintains quality while reducing size

2. **Squoosh** - https://squoosh.app
   - Google's image optimizer
   - Convert to WebP format

3. **ImageOptim (Mac)** - https://imageoptim.com
   - Bulk image optimization

4. **GIMP** - Free Photoshop alternative
   - Resize and edit images
